在 HTML 中，解析 script 元素会应用特殊规则。XHTML 中则没有
        这些规则。这意味着 a < b 语句中的小于号会被解释成一个标签的开始，
        并且由于作为标签开始的小于号后面不能有空格，会导致语法错误。

            有两种解决方法：
                1、小于号都替换成对于的 HTML实体参数（<）。

                2、把所有代码都包含到一个 CDATA 块中。在 XHTML（及 XML）中。
            CDATA 块表示文档可以包含任意文本的区块，其内容不作为标签来解析，
            因此可以在其中包含任意字符，包括小于号，并且不会引发语法错误。

                
                    function compare(a,b){
                        if(a < b){}
